# Adaptive Intake Prompt

Based on the evidence inventory, ask only the questions needed to make the charter initiation-ready.

Prioritize fatal gaps first:

1. Sponsor or approval body
2. Business owner and project owner
3. Scope boundaries
4. Authorized deliverables
5. Success criteria
6. Governance rhythm and escalation path
7. Change-control thresholds

Use plain language. Challenge vague answers. Separate objectives from deliverables.
