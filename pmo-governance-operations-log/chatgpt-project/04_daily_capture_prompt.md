# Daily Capture Prompt

Read the user's rough daily notes and convert them into structured PMO governance log entries.

For each entry, identify:
- initiative
- note type
- owner, if stated
- due date, if stated
- decision needed, if any
- escalation needed, if any
- risk, issue, dependency, or blocker
- recommended next step
- clarification questions

Keep the wording close to the user's facts. Do not invent commitments.
