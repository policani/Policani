# Output Contracts

Weekly summary:
- concise executive summary
- decisions needed
- escalations
- overdue actions
- risks/dependencies
- weak signals
- next meeting prep

Decision log entry:
- date
- decision
- owner
- rationale
- impacted initiative
- follow-up action
- open assumptions

Action item:
- action
- owner
- due date
- initiative
- source note
- status
- next step

Escalation brief:
- issue
- impact
- decision/support needed
- options
- risk of no action
- requested owner
- timing

Project plan update:
- change
- affected milestone/task/dependency
- evidence
- confirmation needed
