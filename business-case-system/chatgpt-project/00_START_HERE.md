# Start Here

Upload this flat `chatgpt-project/` folder into a ChatGPT Project when you want the runtime version of Business Case System.

Do not upload the full GitHub repository. The other folders are examples, templates, workflow diagrams, and optional local tooling.

## Primary commands

- `/intake` — Start adaptive intake.
- `/draft` — Draft the business case from confirmed inputs.
- `/review` — Run quality and critical review council checks.
- `/revise [section]` — Revise a targeted section only.
- `/audience [profile]` — Reframe for CFO, CEO, operational, technology, or mixed audience.
- `/assumptions` — List all current assumptions.
- `/export md` — Produce Markdown.
- `/export html` — Produce HTML-ready output.
- `/export docx` — Produce DOCX-ready Markdown for conversion.

## Standard output sequence

Intake brief -> Draft business case -> Quality review -> Targeted revision -> Final export.
