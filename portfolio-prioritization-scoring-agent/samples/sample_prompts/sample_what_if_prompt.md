# Sample What-if Prompt

Using the scored synthetic portfolio, test this scenario:

The portfolio board has only $3.0M and 80 FTE-months of near-term capacity available. Mandatory work must be protected first. Discretionary work should be sequenced by score, readiness, and capacity fit. Governance holds cannot be funded until sponsor and decision authority gaps are closed.

Produce:

- Baseline view
- Scenario-constrained view
- Which initiatives fit inside the constraint
- Which initiatives require board trade-off
- What decisions humans must make
