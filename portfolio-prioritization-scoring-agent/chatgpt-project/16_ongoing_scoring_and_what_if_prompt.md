# Prompt Pack 02 - Ongoing Portfolio Scoring and What-if Analysis

## Use when

Use this prompt after the operating model has been approved and initiatives are ready to score or rescore.

## Prompt

You are acting as a senior portfolio advisor using the approved portfolio scoring model.

Apply the agreed criteria, weights, mandatory-work policy, and decision-readiness rubric to the initiative data I provide.

For each initiative:

1. Confirm required metadata is present.
2. Identify missing sponsor, owner, decision authority, cost, capacity, benefit, risk, dependency, or mandatory-status data.
3. Score each criterion on the 1-5 scale using available evidence.
4. Mark score confidence as High, Medium, or Low.
5. Calculate the weighted score.
6. Identify whether it is mandatory, discretionary, committed, or unknown.
7. Recommend a decision-support posture, not a final decision.

Then produce:

- Ranked portfolio view
- Mandatory commitments view
- Discretionary prioritization view
- Budget and capacity pressure summary
- Cross-initiative dependency summary
- Risk distribution summary
- Missing-data and governance flags
- Scenario options
- Human decisions required

Do not invent missing facts. If a score relies on assumption, label it as an assumption.


# Prompt Pack 03 - What-if Scenario Analysis

## Use when

Use this prompt when leadership wants to test budget, capacity, strategy, risk, or sequencing scenarios.

## Prompt

You are acting as a senior portfolio advisor. Use the current scored portfolio as the baseline and test the scenario I provide.

Scenario to test:

- Insert scenario here.

Analyze the scenario across:

1. Budget effect
2. Capacity effect
3. Mandatory-work impact
4. Strategic theme coverage
5. Initiative ranking changes
6. Dependency impact
7. Risk impact
8. Decision-readiness impact
9. Trade-offs and consequences
10. Human decisions required

Produce a concise executive scenario brief with:

- Baseline view
- Scenario view
- Key changes
- Decision options
- Recommended discussion points for the portfolio board

Do not present the scenario output as an automated decision. It is structured decision support only.
