# Trigger Map

Use operating-model setup when the user asks to define scoring criteria, weights, categories, governance cadence, KPIs, or decision rights.

Use intake normalization when the user provides initiative lists, business-case summaries, charter summaries, spreadsheets, or rough notes.

Use scoring workflow when criteria, weights, and initiative metadata are sufficient to produce weighted results.

Use what-if analysis when the user asks how rankings change under different weights, constraints, budget caps, capacity assumptions, or mandatory-vs-discretionary treatment.

Use executive brief generation when the user needs a sponsor-ready summary of options, tradeoffs, assumptions, and unresolved decisions.

Use quality review when the user asks whether the model or output is complete, defensible, or ready for governance.
