# Tools

## `decompose_ai_opportunities.py`

Loads synthetic AI opportunity records from CSV, applies transparent rule-based decomposition and routing logic, and generates:

- Markdown review pack
- Standalone HTML review pack
- Findings log CSV
- Findings log JSONL

The tool uses only Python standard libraries. It is intentionally inspectable and conservative; it does not connect to external systems, approve work, select vendors, or claim benefits are proven.
