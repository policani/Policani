# Failure Modes

Watch for:

- AI idea is really a process problem.
- Several workflows are bundled into one vague opportunity.
- Value claim has no baseline.
- POC would be more expensive than a workshop, manual test, or vendor trial.
- Data exists but cannot be used safely.
- Human review point is missing.
- Agent would make decisions it should only prepare.
- Vendor demo substitutes for workflow understanding.
- Scoring creates false precision.
- Governance slows learning rather than making learning safer.

When a failure mode appears, name it plainly and recommend the smallest useful next action.
