# Workflow Shape Review

Classify the work before recommending investment.

Review dimensions:

- Repeatability
- Volume
- Variability
- Judgment intensity
- Exception rate
- Data availability
- Downstream integration
- Human review requirement
- Error cost
- Change-management burden

Useful workflow shapes:

- Bounded repetitive task
- Structured judgment support
- Research and synthesis
- Exception handling
- Decision preparation
- Customer or employee interaction
- Operational coordination
- Content generation
- Data extraction and normalization
- Unclear or unbounded work

Unclear workflow shape is a warning sign. Recommend process mapping or discovery before AI build work.
