# Evaluation Rubric

Score each dimension from 1 to 5.

| Dimension | 1 | 3 | 5 |
| --- | --- | --- | --- |
| Problem clarity | Vague | Plausible | Specific, owned, painful |
| Intelligence decomposition | Missing | Partial | Clear stack across layers |
| Workflow readiness | Unclear | Partly mapped | Bounded and repeatable |
| Value evidence | Unsupported | Directional | Measurable baseline and target |
| Data readiness | Unknown | Partial | Available, approved, usable |
| Control readiness | Unsafe | Needs guardrails | Reviewable and governed |
| Strategic fit | Novelty | Useful | Aligned to priority workflow |
| Proof efficiency | Expensive to learn | Somewhat testable | Cheap credible proof |

Use the score to inform the route, not to replace judgment.
