# Start Here

Use this ChatGPT Project to decompose and evaluate AI opportunity ideas before a team spends time on demos, vendors, proofs of concept, hiring, or delivery planning.

The core question is:

> What intelligence system is this idea actually asking for, and what is the cheapest trustworthy way to prove or disprove it?

Default outputs:

- Concept intake summary
- Intelligence stack decomposition
- Workflow-shape review
- Value and proof review
- Data, risk, and control review
- Route recommendation
- Lightweight experiment plan
- AI opportunity architecture brief
- Decision log entry
