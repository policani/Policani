# File Call Map

When files are provided:

- Use intake forms or CSVs as source data for concept records.
- Use process maps to clarify workflow shape.
- Use policy, security, or governance files to strengthen govern/assure review.
- Use business cases or charters only as evidence; do not treat them as approval.
- Use sample data only as examples, not as real evidence.

If a file contains sensitive data, summarize only what is needed and avoid reproducing confidential details in public-safe outputs.
