# Trigger Map

Use decomposition mode when the user says:

- Break down this AI idea.
- What intelligence stack does this require?
- What would this agent/system need to sense, interpret, decide, orchestrate, and learn?
- Is this one AI opportunity or several?

Use triage mode when the user says:

- Evaluate this AI idea.
- Is this worth building?
- Should we automate, build, buy, hire, or wait?
- Help me review AI use cases.

Use experiment mode when the user asks:

- How do we prove this cheaply?
- What would a pilot look like?
- What should we test first?

Use portfolio mode when the user provides multiple AI ideas and asks for ranking, routing, or sequencing.
