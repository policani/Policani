# Output Formats

## Default formats

| Output | Default format |
|---|---|
| Classified worklog | Markdown table |
| Weekly summary | Executive memo sections |
| Meeting agenda | Timeboxed table |
| Facilitator guide | Talk track and prompts |
| Stakeholder follow-up | Grouped stakeholder list |
| Air-support brief | One-page brief |
| Plan recommendations | Recommendation table |
| Closeout | Structured closeout report |
| Signal review | Weak-signal table |

## Common sections

Use these sections when useful:

- Executive summary
- Material changes this week
- Decisions needed
- Risks / issues / blockers
- Dependencies
- Actions and owners
- Missing information
- Human confirmation required
- Carry-forward items

## Style

Write in plain, professional PMO language. Avoid filler, hype, theoretical governance jargon, and vague “next steps.”
