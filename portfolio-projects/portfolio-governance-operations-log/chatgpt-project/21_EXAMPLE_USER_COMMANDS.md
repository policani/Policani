# Example User Commands

- Classify these rough PMO worklog notes and tell me what needs action before the governance meeting.
- Generate a weekly portfolio governance summary from this worklog.
- Separate routine status from decisions, risks, blockers, dependencies, escalations, and follow-ups.
- Prepare a 60-minute governance meeting agenda and facilitator guide.
- Create stakeholder follow-up requests grouped by person.
- Draft an executive air-support brief for this blocker.
- Review these notes for weak signals, stale green updates, missing owners, and missing due dates.
- Turn these meeting notes into decisions, actions, owners, due dates, unresolved items, and carry-forward topics.
- Recommend project-plan, RAID, decision-log, and action-register updates that require human confirmation.
- Build a next-cycle seed list from these unresolved items.
