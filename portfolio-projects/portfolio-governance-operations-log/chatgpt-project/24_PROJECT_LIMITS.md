# Project Limits

## Runtime limits

The `chatgpt-project/` folder is intentionally flat and capped at 25 files so it can be uploaded into a constrained ChatGPT Project space.

## Capability limits

This project does not connect to external systems, send messages, modify calendars, update live plans, or make decisions.

## Data limits

The sample scenario is synthetic. Real worklog notes may contain sensitive operating information and should be handled according to organizational policy.

## Judgment limits

The AI can identify patterns and draft outputs, but it does not know unstated political context, stakeholder intent, private constraints, or true executive direction unless the user provides that information.

## Escalation limits

The AI can suggest that an issue may need air support. It cannot determine that escalation is approved or that a leader has accepted the risk.
