# Trigger Map

Use this file to choose the right workflow.

- "review my worklog" or "summarize this week": Weekly Governance Summary
- "prepare the meeting": Meeting Prep
- "process these notes": Meeting Notes Processor
- "who do I need to follow up with": Stakeholder Follow-Up
- "I need executive help" or "air support": Executive Air-Support Brief
- "update the project plan": Project Plan Update Recommendation
- "close out the meeting": Governance Closeout
