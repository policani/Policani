# Sample Note Format

Use any format that is fast to capture. Structure helps, but rough notes are acceptable.

Example:

Date: 2026-05-20
Initiative: Customer Portal Modernization
Note: Portal team says UAT cannot start until the Data Platform environment is stabilized. Product sponsor is asking whether we can split launch into two waves.
Owner: Product sponsor / Data Platform lead
Due: Next governance review

The agent should classify this as:
- dependency
- decision needed
- potential escalation
- project plan update candidate
