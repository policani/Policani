# Artifact Ingestion Prompt

Review the artifacts I provide and create an evidence inventory before drafting the charter.

For each artifact, capture:

- Artifact name and type
- Source owner/date/version if available
- Key facts extracted
- Assumptions inferred
- Conflicts or inconsistencies
- Gaps that affect charter readiness
- Charter sections affected
- Confidence level

Do not invent missing facts. If evidence conflicts, flag the conflict and identify who likely needs to decide.
