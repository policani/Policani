# Contributing

Contributions should improve Project Charter Initiation Agent as an agent-assisted project initiation system, not only as a document template.

Useful contributions include:

- Better adaptive interview logic
- Stronger challenge questions
- More realistic sample source artifacts
- Additional charter variants for software, construction, regulatory, operations, product, and engineering contexts
- Improved Markdown, HTML, and DOCX output formatting
- Stronger validation tests
- More precise PMBOK® Guide-aligned project initiation guidance

Do not add vague project-management filler. Every addition should improve authorization, clarity, scope control, governance, risk visibility, decision rights, or execution readiness.
