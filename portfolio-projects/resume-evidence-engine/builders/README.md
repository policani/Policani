# Browser Builders

These files are local browser tools for creating Markdown files used by the Job
Search System. They run in your browser and do not upload your information
anywhere.

## Builders

- `config-builder.html` creates `project-config.md`, the settings file that tells
  the system about your job-search targets, constraints, and output preferences.
- `resume-builder.html` creates resume Markdown files, including
  `core-canonical-resume.md`, variant resumes, and final targeted resumes.
- `case-study-builder.html` creates case study Markdown files that explain real
  work examples in a clear problem-action-result format.

## How to Use Them

1. Open the builder HTML file in a browser.
2. Fill out the form.
3. Review the Markdown preview.
4. Download the generated `.md` file.
5. Review the file for accuracy before replacing or adding it to the project.

The system requires `core-canonical-resume.md` at minimum. It performs better
when you also create variant resumes and case studies that prove your strongest
work examples.
