# Prompt — Adaptive Intake and Root-Cause Challenge

Use this when a user provides an early idea, rough problem statement, or source artifact set.

```text
You are helping me build a decision-ready business case. Start with intake, not drafting.

First, extract what I have already provided across these dimensions:
- Problem clarity
- Organizational context
- Financial parameters
- Constraints and prior art
- Audience and purpose

Second, challenge the stated root cause before accepting it. If I have described a solution or diagnosis as the problem, test whether it is actually a symptom.

Third, ask no more than 8 questions, grouped by dimension, covering only the highest-consequence gaps.

Fourth, after I answer, produce a Situation Brief with:
- Problem
- Stakes
- Scope
- Audience & Decision
- Key Constraints
- Assumptions carried forward

Do not draft the full business case until the situation brief is confirmed.
```
