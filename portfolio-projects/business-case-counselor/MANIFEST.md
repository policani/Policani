# Package Manifest

This package includes:

- A GitHub-ready README, license, contribution guide, and agent instructions.
- A Python CLI for interview, review, and build workflows.
- Business case templates in Markdown, JSON, and CSV.
- Prompt packs for AI-assisted interviewing, artifact synthesis, critical review, and DOCX formatting.
- Documentation for artifact intake, maturity scoring, review-council lenses, and GitHub publishing.
- A sample business case input file, sample evidence artifacts, and generated sample outputs in Markdown and DOCX.
- A smoke test and Makefile for local validation.

The included sample DOCX was rendered and visually checked for layout integrity before packaging.
