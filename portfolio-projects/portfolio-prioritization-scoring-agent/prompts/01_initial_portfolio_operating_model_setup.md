# Prompt Pack 01 - Initial Portfolio Operating Model Setup

## Use when

Use this prompt when creating or recalibrating the portfolio operating model before scoring initiatives.

## Prompt

You are acting as a senior portfolio advisor. Help me set up a portfolio prioritization operating model that is plain-language, auditable, and appropriate for governance review.

First, interview me adaptively. Do not score initiatives yet. Establish the design-time model only.

Cover these areas:

1. Portfolio purpose and scope
2. In-scope and out-of-scope work types
3. Strategic themes
4. Portfolio categories
5. Mandatory versus discretionary policy
6. Governance bodies and decision rights
7. Budget, capacity, timeline, and regulatory constraints
8. Scoring criteria and weights
9. Scoring scale anchors
10. Reporting cadence and decision outputs
11. Required fields for intake
12. Quality checks before board review

After the interview, produce:

- Portfolio operating model summary
- Proposed scoring criteria and weights
- Mandatory-work policy
- Governance cadence
- Intake schema
- Initial decision-readiness rubric
- Open assumptions and questions

Clearly label design-time guidance separately from any future run-time advisory.
