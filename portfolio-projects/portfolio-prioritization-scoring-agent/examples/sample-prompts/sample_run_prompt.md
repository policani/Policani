# Sample Run Prompt

Use the approved default scoring model and score the attached synthetic portfolio intake file.

Inputs:

- `data/default_scoring_criteria.csv`
- `samples/sample_intake_responses/synthetic_portfolio_intake.csv`
- Dummy source summaries in `samples/sample_source_artifacts/`

Produce:

1. Scored portfolio matrix
2. Mandatory commitments view
3. Discretionary prioritization view
4. Budget and capacity summary
5. Risk and dependency summary
6. Missing sponsor, owner, and decision authority flags
7. Human decisions required
8. Self-critique against the quality rubrics

Keep final funding and sequencing decisions positioned as human governance decisions.
