# Hero Banner Lab

Configurable full-screen animated hero banner samples with parallax scroll behavior, pointer response, and exportable JSON settings.

## Evaluate It

- Live demo: https://policani.net/hero-banner-lab.html
- Source repository target: https://github.com/policani/hero-banner-lab

The lab includes three public-safe samples:

- Portfolio Signal: synthetic initiative cards and dependency filaments.
- Decision Gates: AI/LLM-inspired gate columns and attraction-field lines.
- Filament Network: blue data-network nodes, numeric haze, and connection lines.

Each sample uses a 100vh stage so screenshots can be captured at a consistent browser viewport size.

## Status

Standalone public-safe visual prototype. No backend, account, API key, or build system is required.
